## ⚡ Quick Start

### Prerequisites

* Node.js 18+
* npm or pnpm

### Installation

1. Clone the repository

```
git clone https://github.com/gandharvahardikar/portfolio.git
cd portfolio
```

2. Install dependencies

```
npm install
```

3. Start the development server

```
npm run dev
```

4. Open the configuration file and customize your details

```
src/data/resume.tsx
```

## 🎨 Customization

All your personal information lives in a single file.

src/data/resume.tsx

Update your name, bio, projects, social links, and more in one place.
